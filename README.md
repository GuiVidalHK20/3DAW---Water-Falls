# 3DAW---Water-Falls

Código em PHP com o objetivo de criar um site com um jogo de perguntas e respostas numa corporação. 
  Apresenta cadastro de funcionário, Perguntas de múltipla escolha, Perguntas e Respostas de Texto e Como listar, excluir ou alterar uma pergunta.
    Programado usando Xampp, PHP, HTML e VSCode.
